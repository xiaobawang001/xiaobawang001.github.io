# Docker镜像源配置

## 1. 镜像源监控

> **1panel监控**  [https://status.1panel.top/](https://status.1panel.top/)
> **容器镜像监控**  [https://status.anye.xyz/](https://status.anye.xyz/)

## 2. 更换镜像源

### 2.1 编辑配置文件

```bash
sudo vim /etc/docker/daemon.json
```

### 2.2 写入以下内容

```json
{
  "registry-mirrors": [
    "https://docker.xuanyuan.me",
    "https://docker.1ms.run",
    "https://docker.m.daocloud.io",
    "https://docker.1panel.live"
  ]
}
```

### 3. 重启docker服务

```sh
sudo systemctl daemon-reload
sudo systemctl restart docker
```
