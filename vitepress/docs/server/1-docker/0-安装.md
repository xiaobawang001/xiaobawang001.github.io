# Docker安装

## 1. 官方安装脚本
```sh
curl -fsSL get.docker.com -o get-docker.sh && sh get-docker.sh
```

## 2. 一键安装脚本（root身份）

> **官网：[https://linuxmirrors.cn/](https://linuxmirrors.cn/)**
> **说明**：集成安装 Docker Engine 和 Docker Compose，支持选择或更换软件源（Docker 软件仓库）以及镜像仓库、安装指定版本、重装等功能，支持 ARM 架构.脚本参考官方文档使用系统包管理工具进行安装，不存在兼容性、安全性等问题，可安装的版本由 Docker CE 仓库决定

```sh
# 官网（推荐）
bash <(curl -sSL https://linuxmirrors.cn/docker.sh)

# github
bash <(curl -sSL https://raw.githubusercontent.com/SuperManito/LinuxMirrors/main/DockerInstallation.sh)

# gitee(推荐)
bash <(curl -sSL https://gitee.com/SuperManito/LinuxMirrors/raw/main/DockerInstallation.sh)

# gitcode
bash <(curl -sSL https://raw.gitcode.com/SuperManito/LinuxMirrors/raw/main/DockerInstallation.sh)

# jsDelivr(CDN)
bash <(curl -sSL https://cdn.jsdelivr.net/gh/SuperManito/LinuxMirrors@main/DockerInstallation.sh)

# 腾讯云EO(CDN)
bash <(curl -sSL https://edgeone.linuxmirrors.cn/docker.sh)

# daocloud
sudo curl -sSL https://get.daocloud.io/docker | sh
```
