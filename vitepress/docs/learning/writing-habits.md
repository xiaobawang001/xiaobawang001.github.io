# Markdown 写作习惯

> 2025-06-15

1. 每篇笔记一个文件，用 front matter 标注标题和日期
2. 代码块标注语言以启用高亮
3. 短笔记直接写，长文先列大纲再填充

## 文章模板

```markdown
---
title: 文章标题
date: 2025-06-15
---

# 正文标题

正文内容...
```

## 目录组织建议

- 按主题建文件夹，如 `dev/`、`learning/`
- 在 `config.ts` 的 sidebar 里配置可折叠的分类树
- 宽屏下右侧自动显示当前文章的页内目录
